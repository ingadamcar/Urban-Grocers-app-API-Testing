# Urban Grocers API Testing

En este proyecto se diseñaron y ejecutaron pruebas de API manuales para validar principalmente códigos de estado utilizando Postman siguiendo estrictamente los requerimientos dados para el proyecto. Se utilizó JIRA para reportar los bugs.

In this project, manual API tests were designed and executed to primarily validate status codes using Postman, strictly adhering to the project’s specified requirements. JIRA was used to report the bugs.
